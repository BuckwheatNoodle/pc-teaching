SECTION 02
🤖 AIチャットの使い方ルール

ChatGPTやClaudeなどのAIチャットは、プログラミング学習のすごい味方になる。
でも使い方を間違えると、逆に成長できなくなる。ルールを決めておこう。

3つのルール

🥇 ルール1: まず自分で15分考える

すぐAIに聞くのはNG。まずは自分でコードを書いてみよう。
エラーが出ても、エラーメッセージを読んで自分で直してみる。
15分やっても解決しないときに初めてAIに聞く。

🥈 ルール2: 「答え」ではなく「ヒント」を聞く

×「じゃんけんゲームのコードを書いて」
○「じゃんけんゲームで、コンピュータの手をランダムに出す方法がわかりません。ヒントをください」
完成品をもらうのではなく、わからない部分だけ質問する。

🥉 ルール3: もらった答えを必ず理解する

AIのコードをコピペして動いた！…で終わらない。
「なぜこう書くのか」を自分の言葉で説明できるまで理解する。
わからなければ「この部分を初心者向けに説明して」と追加で聞こう。

良い質問と悪い質問

❌ 悪い質問

「Pythonでじゃんけんゲーム作って」
「このエラー直して」
「数当てゲームのコード全部ください」

→ 自分で考えていない。成長しない。

✅ 良い質問

「random.choice の使い方を教えて」
「このエラーはどういう意味？: NameError: name 'x' is not defined」
「whileループの中でbreakを使うとどうなる？」

→ 具体的で、学ぶ意欲がある。

SECTION 08
💬 AIに質問する練習

実際にAIチャットを使ってみよう！
ここでは「いい質問の仕方」を練習します。

場面1: エラーが出たとき

🤖 質問の例
Python で以下のエラーが出ました。初心者なので原因を教えてください。

TypeError: can only concatenate str to str

自分のコード:
print("スコア: " + score)

score は数値の変数です。
このエラーは、文字列（str）と数値（int）を + でつなごうとしたときに出ます。

数値を文字列に変換するには str() を使います：
print("スコア: " + str(score))

場面2: やり方がわからないとき

🤖 質問の例
じゃんけんゲームを3回勝負にしたいです。
今は1回だけ動くコードが書けています。
3回繰り返して、最後に「2勝1敗」のように表示するには、
どういう考え方で書けばいいですか？
コード全体ではなく、考え方のヒントを教えてください。
いい質問です！考え方のヒント：
① 今の1回分のコードを for ループで3回繰り返す
② 勝ち数を数える変数（例: win_count = 0）を用意する
③ 勝つたびにその変数に1を足す
④ ループが終わったら結果を表示

まずはこの流れで書いてみてください！

場面3: コードを理解したいとき

🤖 質問の例
以下のコードの意味がわかりません。1行ずつ初心者向けに説明してもらえますか？

result = [x * 2 for x in range(5)]
これは「リスト内包表記」という書き方です。
普通に書くとこうなります：

result = []
for x in range(5): ← 0〜4を繰り返す
result.append(x * 2) ← 各数を2倍にして追加

結果: [0, 2, 4, 6, 8]
これを1行で書いたのがさっきのコードです！
🎯 実際にやってみよう
ChatGPT（chat.openai.com）やClaude（claude.ai）を開いて、以下を試してみよう：

① 「Python の random.shuffle はどういう動きをしますか？簡単な例で教えてください」と聞く
② 返ってきた説明を自分の言葉で誰かに説明できるか確認する
③ サンプルコードを手で打って動かしてみる（コピペ禁止！）

手で打つことで、コードが頭に入るよ！
⚠️ AIの回答は100%正しいとは限らない！
たまに間違った情報を教えることもあります。
おかしいと思ったら「本当にそうですか？」と確認したり、実際にコードを動かして確かめよう。
