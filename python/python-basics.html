SECTION 01
🐍 Pythonってなに？
Python（パイソン）は、世界で一番人気のあるプログラミング言語のひとつ。
名前はニシキヘビ🐍ではなく、イギリスのコメディ番組「モンティ・パイソン」が由来です。
📖
読みやすい
英語に近い文法で、初心者でもわかりやすい
🌍
なんでもできる
Web・AI・ゲーム・データ分析など用途が幅広い
🤝
みんな使ってる
Google, YouTube, Instagram なども Python を活用
🆓
無料で使える
ダウンロードもインストールも完全無料！
💡 「プログラミングをはじめたいけど、何からやればいい？」という人には Python がおすすめ。読みやすくて、少ないコードで色々なことができるよ！
SECTION 02
💻 インストールしよう
Windows 11 に Python をインストールする手順を説明します。
1
Python 公式サイトにアクセス
ブラウザで python.org を開きます。
上のメニューから「Downloads」をクリックすると、Windows用のダウンロードボタンが表示されます。
2
「Download Python 3.x.x」ボタンをクリック
大きな黄色いボタンが表示されるので、それをクリック。
インストーラー（.exe ファイル）がダウンロードされます。
3
インストーラーを実行
ダウンロードしたファイルをダブルクリックして開きます。
⚠️ 超重要！インストーラー画面の下にある
「Add python.exe to PATH」にチェックを入れる！
これを忘れると、後からコマンドで Python が使えなくなります。
4
「Install Now」をクリック
推奨設定のままで大丈夫です。インストールが始まるので少し待ちましょう。
5
「Setup was successful」と出たら完了！
「Close」を押してインストーラーを閉じましょう。

インストールできたか確認しよう

キーボードで Win + R を押して「ファイル名を指定して実行」を開き、
cmd と入力して Enter。黒い画面（コマンドプロンプト）が開きます。

💻 コマンドプロンプト
C:\Users\あなた> python --version Python 3.12.4 ↑ バージョン番号が出たら成功！
⚠️ 'python' は認識されていません と出た場合は、ステップ3で PATH にチェックを入れ忘れた可能性があります。もう一度インストーラーを実行して「Modify」→「Add to PATH」を有効にしましょう。

SECTION 05
🧠 プログラミングの基礎概念
コードを書く前に、プログラミングの考え方を理解しておこう。
難しそうに見えるけど、実は日常生活でやっていることと同じです。

プログラムとは「手順書」

たとえばカップラーメンを作る手順：

  1. フタを開ける
  2. お湯を入れる
  3. 3分待つ
  4. 食べる

プログラムもこれと同じ！コンピュータにやってほしいことを、順番に書いていくだけです。

💡 コンピュータは人間と違って「空気を読む」ことができません。だから正確に、細かく指示を書く必要があるよ。

プログラミングの3つの基本構造

どんなに複雑なプログラムも、この3つの組み合わせでできています：

➡️
順次（じゅんじ）
上から順番に実行する。
基本中の基本！
🔀
分岐（ぶんき）
条件によって処理を変える。
「もし〜なら」
🔁
反復（はんぷく）
同じことを繰り返す。
「〜回くり返す」

日常の例で考えてみると：

📖 日常生活のプログラム
【順次】朝の準備 起きる → 顔を洗う → 着替える → 朝ごはん 【分岐】天気で判断 もし 雨なら → 傘を持つ そうでなければ → そのまま出発 【反復】宿題 問題が残っている間 → 1問解く をくり返す

その他の重要な考え方

📦
変数
データに名前をつけて保存する箱
🏷️
データ型
数字・文字など、データの種類
⚙️
関数
まとめた処理に名前をつけて再利用
🐛
デバッグ
バグ（間違い）を見つけて直すこと
💡 これから先のセクションで、これらの概念を Python のコードで実際に体験していくよ！

データ型とは？

変数に入れるデータには種類（型）があります。

📖 主なデータ型
文字列（str） "こんにちは" "Python" "123" → 文字の並び。ダブルクォーテーションで囲む 整数（int） 42 -10 0 2025 → 小数点のない数字 小数（float） 3.14 -0.5 165.5 → 小数点のある数字 真偽値（bool） True False → 「はい」か「いいえ」の2択
💡 数字の "123" と数値の 123 は違う！
"123" は文字列（足し算すると結合される → "123" + "456" → "123456"）
123 は整数（足し算すると計算される → 123 + 456 → 579）

変数の名前のルール

  • 英字・数字・アンダースコア（_）が使える
  • 数字で始めてはダメ（1name はNG、name1 はOK）
  • 予約語（if, for など Python が使う単語）は使えない
  • わかりやすい名前をつけよう！（x より score の方がわかりやすい）

SECTION 08
🔀 条件分岐（if文）

プログラムに「判断」させることができます。
「もし〜なら、〇〇する。そうでなければ、△△する。」という条件分岐の命令です。

📖 if文の書き方
if 条件: 条件が正しいとき実行される処理 elif 別の条件: ← 何個でも追加できる 別の条件が正しいとき else: どれも正しくないとき
⚠️ インデント（字下げ）が超重要！
if の中の処理はスペース4つで下げます。これがないとエラーになります。
Python はインデントで「どこまでが if の中か」を判断しています。

比較演算子

📖 条件に使える記号
== 等しい age == 14 != 等しくない age != 20 > より大きい score > 80 < より小さい score < 50 >= 以上 score >= 80 <= 以下 score <= 100

SECTION 10
📝 リスト

複数のデータをまとめて管理したいとき、リストを使います。

📖 リストの基本
作り方: fruits = ["りんご", "バナナ", "みかん"] 取り出し: fruits[0] → "りんご" # 0番目から始まる！ 追加: fruits.append("ぶどう") 個数: len(fruits) → 4 削除: fruits.remove("バナナ")
💡 番号は0から始まる！これはプログラミング全般のルール。
1番目ではなく「0番目」が最初の要素です。最初はちょっと混乱するけど、すぐ慣れるよ！

SECTION 11
⚙️ 関数

同じ処理を何度も使いたいとき、関数にまとめると便利です。
「自分だけのオリジナル命令」を作るようなものです。

📖 関数の書き方
def 関数名(引数): 処理 return 戻り値 ← 結果を返す（省略可） # 使い方 結果 = 関数名(値)

def は「define（定義する）」の略。
引数（ひきすう）は関数に渡すデータ、戻り値（もどりち）は関数が返すデータです。
