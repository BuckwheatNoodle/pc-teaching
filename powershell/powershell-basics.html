SECTION 01
⬛ CLIってなに？
ふだんパソコンを使うとき、マウスでアイコンをクリックして操作するよね。
これを GUI（ジーユーアイ）= グラフィカル・ユーザー・インターフェース と言います。

一方、CLI（シーエルアイ）= コマンドライン・インターフェース は、文字（コマンド）を打ってパソコンを操作する方法です。

🖱️
GUI
マウスでクリックして操作
見た目でわかりやすい
⌨️
CLI
キーボードでコマンドを打つ
速くて正確、自動化もできる

なぜ CLI を学ぶの？

  • プログラミングに必須 — Python の実行、Git の操作、サーバー管理…全部 CLI
  • 速い — マウスでポチポチするより、コマンド1行の方が圧倒的に速い場面がある
  • 自動化できる — 同じ操作を100回やるなら、コマンドを書けば一瞬
  • かっこいい — 映画のハッカーみたいでテンション上がる！

PowerShell ってなに？

PowerShell は Windows に最初から入っている CLI ツール。
古い「コマンドプロンプト」の進化版で、もっと多くのことができます。

💡 Mac には「ターミナル」、Linux には「Bash」がある。PowerShell は Windows 版の CLI と覚えよう���

さっそくコマンドを打ってみよう！

今いる場所を確認する — pwd

⬛ PowerShell
PS C:\Users\あなた> ls ディレクトリ: C:\Users\あなた Mode LastWriteTime Length Name ---- ------------- ------ ---- d----- 2025/04/01 10:00 Desktop d----- 2025/04/01 10:00 Documents d----- 2025/04/01 10:00 Downloads d----- 2025/04/01 10:00 Music d----- 2025/04/01 10:00 Pictures

SECTION 05
📁 ファイル・フォルダ操作

GUI ではマウスで右クリック → 「新しいフォルダ」としていた操作を、コマンドでやってみよう。

フォルダを作る — mkdir

⬛ PowerShell
PS C:\Users\あなた\Desktop> mkdir test-folder ディレクトリ: C:\Users\あなた\Desktop Mode LastWriteTime Length Name ---- ------------- ------ ---- d----- 2025/04/04 15:30 test-folder # フォルダの中にさらにフォルダも一気に作れる PS ...\Desktop> mkdir test-folder\sub1\sub2

ファイルを作る — New-Item

⬛ PowerShell
# 空のファイルを作る PS ...\Desktop> New-Item hello.txt # 中身を書いてファイルを作る PS ...\Desktop> "こんにちは！" | Out-File memo.txt

コピー — Copy-Item (cp)

⬛ PowerShell
# ファイルをコピー PS ...\Desktop> cp hello.txt hello-backup.txt # フォルダごとコピー（-Recurse が必要） PS ...\Desktop> cp test-folder test-folder-backup -Recurse

移動・名前変更 — Move-Item (mv)

⬛ PowerShell
# ファイルを別のフォルダに移動 PS ...\Desktop> mv hello.txt test-folder\ # ファイル名を変更（移動先の名前を変えるだ��） PS ...\Desktop> mv memo.txt メモ.txt

削除 — Remove-Item (rm)

⬛ PowerShell
# ファイルを削除 PS ...\Desktop> rm hello-backup.txt # フォルダを中身ごと削除（-Recurse をつける） PS ...\Desktop> rm test-folder-backup -Recurse
⚠️ rm で消したファイルはゴミ箱に入りません！
完全に消えてしまうので、削除するときは慎重に。
大事なファイルは事前にバックアップを取ろう。
🎯 やってみよう
デスクトップで以下を試してみよう（最後にお掃除も忘れずに！）：
① mkdir practice でフォルダ作成
② cd practice で中に入る
③ "Hello!" | Out-File test.txt でファイル作成
④ ls で確認
⑤ cp test.txt test2.txt でコピー
⑥ cd .. で戻って rm practice -Recurse でお掃除
SECTION 06
👀 ファイルの中身を見���

テキストファイルの中身をわざわざメモ帳で開かなくても、コマンドで見ることができます。

全部表示 — cat (Get-Content)

⬛ PowerShell
PS ...\Desktop> cat memo.txt 今日やること： 1. 数学の宿題 2. 英語の単語テスト 3. プログラミングの練習

最初の数行だけ — Get-Content -Head

⬛ PowerShell
# 最初の3行だけ表示 PS ...\Desktop> Get-Content memo.txt -Head 3 今日やること： 1. 数学の宿題 2. 英語の単語テスト

最後の数行だけ — Get-Content -Tail

⬛ PowerShell
# 最後の2行だけ表示 PS ...\Desktop> Get-Content memo.txt -Tail 2 2. 英語の単語テスト 3. プログラミングの練習

文字を検索する — Select-String

⬛ PowerShell
# ファイルの中から「英語」を含む行を探す PS ...\Desktop> Select-String "英語" memo.txt memo.txt:3:2. 英語の単語テスト # フォルダ内の全 .txt ファイルから検索 PS ...\Desktop> Select-String "TODO" *.txt
💡 Select-String は GUI で言えば Ctrl+F（検索）のようなもの。
ただし、複数ファイルを一括検索できるのが CLI の強み！
SECTION 07
⚡ 便利テクニッ���

CLI を使いこなすための必須テクニックを紹介！

Tab補完 — 最重要テクニック！

フォルダ名やファイル名を全部打つ必要はありません。途中まで打って Tab を押すと自動で補完されます！

⬛ PowerShell
# 「Doc」まで打って Tab を押すと… PS C:\Users\あなた> cd Doc[Tab] PS C:\Users\あなた> cd .\Documents\ ← 自動で補完された！ # 候補が複数あるときは Tab を何度か押すと切り替わる
⚠️ Tab 補完を使わずに長いパスを手打ちするのはミスの元！
CLI 上級者ほど Tab を多用します。どんどん使おう。

履歴 — 上下キー

↑ �� キーで過去に打ったコマンドを呼び出せます。同じコマンドを何度も打つ必要なし！

�� PowerShell
# 過去のコマンド一覧を見る PS C:\Users\あなた> Get-History Id CommandLine -- ----------- 1 pwd 2 ls 3 cd Desktop 4 mkdir test-folder

コマンドの中断 — Ctrl + C

実行中のコマンドを止めたいとき、Ctrl + C で強制停止できます。

コマンドのヘルプ — Get-Help

⬛ PowerShell
# コマンドの使い方を調べる PS C:\Users\あなた> Get-Help Get-ChildItem # 短く使い方の例だけ見る PS C:\Users\あなた> Get-Help Get-ChildItem -Examples
💡 覚えるコツ：全コマンドを暗記する必要はない！
Tab で補完、↑↓ で履歴、困ったら Get-Help。
この3つだけで十分やっていけ��！
SECTION 08
🔗 パイプと��ィルタ

CLI の真の力は、コマンドをつなげることにあります。

パイプ | とは？

あるコマンドの出力を、別のコマンドの入力として渡す仕組みです。
水道管（パイプ）のように、データが流れていくイメージ。

📖 パイプのイメージ
コマンドA | コマンドB | コマンドC ↓ ↓ ↓ データを出す → 加工する → さらに加工する

実際に使ってみよう

⬛ PowerShell
# ls の結果から .txt ファイルだけ表示 PS ...\Desktop> ls | Where-Object { $_.Name -like "*.txt" } # ファイルを更新日時の新しい順に並べる PS ...\Desktop> ls | Sort-Object LastWriteTime -Descending # ファイルの数を数える PS ...\Desktop> ls | Measure-Object Count : 12 # ファイル名だけ表示 PS ...\Desktop> ls | Select-Object Name

出力をファイルに保存

⬛ PowerShell
# 結果をファイルに書き出す PS ...\Desktop> ls | Out-File filelist.txt # > でも同じことができる（ショートカット） PS ...\Desktop> ls > filelist.txt # >> で追記（上書きせず末尾に追加） PS ...\Desktop> echo "追加のメモ" >> memo.txt
💡 パイプは CLI 最大の武器！
小さなコマンドを組み合わせて、複雑な操作を1行で実現できる。
まるでレゴブロックを組み立てるような感覚！
