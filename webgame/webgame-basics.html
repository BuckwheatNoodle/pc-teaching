GAME 03
🔨 もぐらたたき

出てきたもぐらをクリック！15秒で何匹たたけるか？

mogura.html
プレビュー
💡 新しい知識:
・display: grid — CSS で格子状にレイアウトする（穴の配置に使用）
・document.createElement — JavaScript でHTMLの要素を動的に作る
・スコアが上がるほど速くなる仕組み: Math.max(400, 1000 - score * 30)
🚀 改造してみよう
・穴を 4×4 の16個に増やしてみよう（grid-template-columns を変える）

・たまに爆弾（💣）が出て、クリックするとスコアが減るようにしよう

プレビュー（← →キーでカゴを操作！プレビュー内をクリックしてからキーを押してね）

SECTION 07
💻 自分のPCで動かす方法

このページのエディタで試すのもいいけど、自分のPCでファイルを作ると自由度が上がる！

  1. メモ帳やVS Codeを開く
  2. 上のコードを貼り付ける
  3. .html で保存する（例: game.html）
  4. 保存したファイルをダブルクリック → ブラウザで開く！
💡 これだけ！サーバーもインストールも不要。ファイルを保存してダブルクリックするだけで動くのが HTML の最大の魅力。
改造するときは、ファイルを編集 → 保存 → ブラウザで再読み込み（F5）のくり返し！
