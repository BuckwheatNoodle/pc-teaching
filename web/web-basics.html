SECTION 01
🌍 Webページの仕組み

今見ているこのページも、全部 HTML・CSS・JavaScript の3つでできている。

🏗️
HTML
ページの中身と構造
文字、画像、ボタンなど
「何を置くか」
🎨
CSS
ページの見た目
色、大きさ、配置
「どう見せるか」
⚡
JavaScript
ページの動き
クリック、アニメーション
「どう動かすか」

たとえて言うと：

🏠
HTML = 骨組み
柱、壁、窓の配置
🖌️
CSS = 内装
壁の色、家具の配置
💡
JS = 電気設備
スイッチ、自動ドア
💡 この章ではこの3つの基本を順番に学んでいくよ。
全部のエディタでコードを変えて「プレビュー」を押せば、すぐ結果が見える！

SECTION 08
💻 自分のPCで作る方法
  1. メモ帳やVS Codeを開く
  2. コードを書く（または上のコードを貼り付ける）
  3. .html で保存する（例: profile.html）
  4. 保存したファイルをダブルクリック → ブラウザで開く！
💡 これだけ！保存してダブルクリックするだけでWebページが完成。
編集したら保存 → ブラウザで F5（再読み込み）で反映されるよ。
