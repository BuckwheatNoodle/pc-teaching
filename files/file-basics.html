SECTION 01
📄 ファイルとフォルダってなに？

パソコンの中にあるデータは、すべてファイルとして保存されています。

📄 ファイル

データそのもの。写真、文書、音楽、動画、プログラム… 全部ファイル。
現実世界の「紙」や「写真」にあたる。

📁 フォルダ

ファイルを整理するための「入れ物」。
フォルダの中にフォルダも入れられる。
現実世界の「引き出し」や「本棚」にあたる。

現実世界でたとえると

📖 パソコンの中身のイメージ
🏠 C:\Users\あなた\（自分の部屋）
　├── 📁 Desktop（机の上）← すぐ使うものを置く
　├── 📁 Documents（書類の棚）← レポートやメモ
　├── 📁 Downloads（郵便受け）← ダウンロードしたもの
　├── 📁 Pictures（写真アルバム）
　├── 📁 Music（CDラック）
　└── 📁 Videos（DVDの棚）
💡 パソコンを使いこなすには、このファイルとフォルダの関係を理解することが一番大事！
「自分のデータがどこにあるか」を常に把握しておこう。

SECTION 03
✨ 作成・名前変更

新しいフォルダを作る

1
フォルダを作りたい場所で空白部分を右クリック
2
「新規作成」→「フォルダー」を選ぶ
3
名前を入力して Enter
Ctrl + Shift + N
新しいフォルダ（ショートカット）

新しいファイルを作る

1
空白部分を右クリック
2
「新規作成」→ 作りたい種類を選ぶ（テキスト文書、Wordなど）
3
名前を入力して Enter

名前を変更する

ファイルやフォルダの名前を変える方法はいくつかあります：

  • ファイルを選択 → F2 キー（おすすめ！）
  • ファイルを右クリック → 「名前の変更」
  • ファイル名をゆっくり2回クリック（ダブルクリックではなく）
⚠️ 名前を変えるときに拡張子（.txt, .jpg など）を消さないように注意！
拡張子を消すとファイルが開けなくなることがあります。（拡張子については後で詳しく説明）
🎯 やってみよう
① デスクトップで Ctrl + Shift + N で新しいフォルダを作る
② 名前を「練習フォルダ」にする
③ その中に右クリック → 「新規作成」→「テキスト文書」でファイルを作る
④ F2 で名前を「メモ.txt」に変更
SECTION 04
📋 コピー・移動・削除

コピー & 貼り付け

ファイルを別の場所にコピーする方法：

1
コピーしたいファイルを選択して Ctrl + C
2
コピー先のフォルダを開いて Ctrl + V
💡 元のファイルはそのまま残る。同じファイルが2つになるイメージ！

切り取り & 貼り付け（移動）

1
移動したいファイルを選択して Ctrl + X
2
移動先のフォルダを開いて Ctrl + V
💡 元の場所からは消える。引っ越しのイメージ！

ドラッグ & ドロップ

マウスでファイルを掴んで移動する方法もあります：

  • 同じドライブ内でドラッグ → 「移動」になる
  • 別のドライブにドラッグ → 「コピー」になる
  • Ctrl を押しながらドラッグ → 強制的に「コピー」
  • Shift を押しながらドラッグ → 強制的に「移動」

削除

  • Delete キー → ゴミ箱へ移動（復元できる）
  • Shift + Delete → 完全に削除（復元できない！）
⚠️ Shift + Delete は本当に消えます！大事なファイルには絶対使わないこと。
普通の Delete なら、ゴミ箱から元に戻せるので安心。

複数のファイルを選ぶ方法

Ctrl + クリック
バラバラに1つずつ追加選択
📏
Shift + クリック
最初〜最後まで一括選択
☑️
Ctrl + A
全部選択
🎯 やってみよう
① さっき作った「練習フォルダ」の中にテキストファイルを3つ作る
② Ctrl + クリックで2つだけ選択
③ Ctrl + C でコピー
④ デスクトップに戻って Ctrl + V で貼り付け

SECTION 05
🏷️ 拡張子を知ろう

ファイル名の最後にある .txt や .jpg が拡張子。
これは「このファイルは何のデータか」をパソコンに教えるラベルのようなものです。

📁 拡張子の見え方
📝 レポート.txt 2 KB テキスト文書
🖼️ 写真.jpg 3.2 MB JPEG画像
📄 宿題.docx 45 KB Word文書
🐍 hello.py 1 KB Pythonファイル

よく使う拡張子一覧

拡張子種類用途
.txtテキストメモや設定ファイル。一番シンプル
.docxWord文書レポートや文書。装飾や図も入る
.xlsxExcel表計算。成績管理やグラフ
.pptxPowerPointプレゼンテーション・スライド
.pdfPDF文書誰でも同じ見た目で読める文書
拡張子種類用途
.jpg / .jpeg画像写真に使われる。ファイルが軽い
.png画像イラストやスクショ。透明も使える
.gif画像動くアニメーション画像
.mp3音声音楽ファイル
.mp4動画動画ファイル。YouTubeもこれ
拡張子種類用途
.pyPythonPython のプログラム
.htmlWebウェブページ
.zip圧縮ファイルをまとめて圧縮したもの
.exe実行アプリケーション本体

拡張子を表示する設定（重要！）

Windows の初期設定では拡張子が隠されていることがあります。
必ず表示するように設定しましょう。

1
エクスプローラーを開く（Win + E）
2
上のメニューから「表示」をクリック
3
「表示」→「ファイル名拡張子」にチェックを入れる
⚠️ 拡張子の表示は必ずONにしよう！
拡張子が見えないと「写真だと思って開いたら実はウイルス（.exe）だった」という危険があります。
プログラミングでも拡張子を間違えるとファイルが正しく動きません。
SECTION 06
🗺️ パスを理解しよう

パス（Path）とは、ファイルやフォルダの「住所」のこと。
パソコンの中のどこにデータがあるかを正確に示します。

パスの読み方

📍 パスの構造
C:\Users\あなた\Documents\レポート.docx
■ Cドライブ（パソコン本体）
■ ユーザーフォルダ
■ あなたのフォルダ
■ ドキュメントフォルダ
■ ファイル本体

絶対パスと相対パス

🏠 絶対パス

一番上（C:\）からの完全な住所。
C:\Users\あなた\Documents\レポート.docx
「東京都渋谷区○○町1-2-3」のようなもの。どこにいても場所がわかる。

📍 相対パス

今いる場所からの道順。
Documents\レポート.docx
「この道をまっすぐ→右」のようなもの。今いる場所によって変わる。

パスを確認する方法

1
エクスプローラーのアドレスバーをクリックするとパスが表示される
2
ファイルを右クリック→「パスのコピー」で完全なパスをコピーできる
💡 パスの理解はプログラミングで超重要。
「ファイルが見つからない」系のエラーは、ほとんどの場合パスの間違いが原因！
PowerShell の cd や Python の open() でもパスを使うよ。
SECTION 07
🧹 ファイル整理術

ファイルが増えてくると、あちこちに散らばってカオスになりがち。最初からルールを決めておこう！

整理の基本ルール

1
デスクトップにファイルを溜めない
デスクトップは「作業台」。作業が終わったらドキュメントに移動しよう。
2
フォルダで分類する
「学校」「プログラミング」「写真」など、種類ごとにフォルダを作ろう。
3
わかりやすい名前をつける
×「無題」「新しいフォルダー (2)」
○「2025_数学レポート」「夏休み_写真」
4
ダウンロードフォルダを定期的にお掃除
ダウンロードフォルダは放っておくと大量のファイルが溜まります。

おすすめのフォルダ構成

📁 ドキュメント — おすすめ構成
📁 ドキュメント
　├── 📁 学校
　│　　├── 📁 数学
　│　　├── 📁 英語
　│　　└── 📁 理科
　├── 📁 プログラミング
　│　　├── 📁 python-practice
　│　　└── 📁 web-practice
　├── 📁 写真
　│　　├── 📁 2025-04_旅行
　│　　└── 📁 2025-03_卒業式
　└── 📁 その他

ファイル名のつけ方のコツ

  • 日付を先頭に → 2025-04-01_レポート.docx（自動で日付順に並ぶ！）
  • 使えない文字に注意 → \ / : * ? " < > | は使えない
  • スペースの代わりにハイフンかアンダースコア → my-report や my_report
  • バージョンをつける → レポート_v1.docx レポート_v2.docx
💡 プログラミングの世界ではフォルダ名・ファイル名は英語（半角英数字）が基本。
日本語だとエラーの原因になることがあるので、プログラミング用フォルダは英語で名前をつけよう！
SECTION 08
📦 圧縮と展開（ZIP）

複数のファイルをまとめて1つにしたり、ファイルサイズを小さくするのが圧縮。
メールで送るときや、ネットからダウンロードしたファイルでよく使います。

📦 圧縮

複数のファイルを1つの .zip ファイルにまとめる。
「段ボールに荷物を詰める」イメージ。

📂 展開（解凍）

.zip ファイルから元のファイルを取り出す。
「段ボールを開けて中身を出す」イメージ。

圧縮する方法

1
まとめたいファイルをすべて選択する
2
選んだファイルを右クリック
3
「ZIPファイルに圧縮する」を選ぶ

展開（解凍）する方法

1
.zip ファイルを右クリック
2
「すべて展開」を選ぶ
3
展開先を選んで「展開」をクリック
⚠️ ZIP の中身を直接編集しないこと！
ZIP をダブルクリックすると中身が見えますが、これは「覗いている」だけ。
編集したいなら必ず「すべて展開」してからにしよう。展開せずに編集すると変更が消えることがあります。
🎯 やってみよう
① 「練習フォルダ」の中にファイルを2〜3個作る
② 全部選んで右クリック → 「ZIPファイルに圧縮する」
③ 作られた .zip ファイルを別の場所に移動
④ 右クリック → 「すべて展開」で中身を取り出す

SECTION 10
🔧 トラブル対処法

よくあるファイル操作のトラブルと対処法をまとめました。

「ファイルが見つからない！」

🔍
エクスプローラーの検索機能を使おう
エクスプローラー右上の検索バーにファイル名を入力。
Win + S の Windows 検索でも探せるよ。

「間違えてファイルを消しちゃった！」

♻️
ゴミ箱を確認しよう
デスクトップの「ゴミ箱」を開いて、ファイルを右クリック → 「元に戻す」。
Ctrl + Z（元に戻す）でも復元できることがあるよ。

「ファイルが開けない！」

🏷️
拡張子を確認しよう
・拡張子が消えていないか？ → 名前を変更して拡張子を付け直す
・対応するアプリがインストールされているか？ → 右クリック → 「プログラムから開く」

「このファイルは使用中です」と出る

🔒
そのファイルを開いているアプリを閉じよう
Word で開いているドキュメントは、Word を閉じないと移動・削除できない。
どのアプリが使っているかわからないときは、一度パソコンを再起動してみよう。

「アクセスが拒否されました」と出る

🛡️
Windows のシステムファイルは触れない
C:\Windows や C:\Program Files の中は、Windows が使う大事なファイル。
これらは消したり変更したりしないように保護されています。触らないようにしよう！
💡 困ったときの3ステップ：
① まず Ctrl + Z（元に戻す）を試す
② ゴミ箱を確認する
③ それでもダメなら、大人に相談しよう！
